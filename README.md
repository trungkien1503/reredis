# Reredis

Custorm redis gem

## Installation

Add this line to your application's Gemfile:

    gem 'reredis', '~> 0.0.1'

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install reredis

## Usage
  A custorm gem redis- it include redis object and some sample counter for model : photo,user using Redis
